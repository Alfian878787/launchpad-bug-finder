Launchpad bug finder
===

A script to retrieve a list of bugs in the UX team, between two dates.

Dependencies
---

Install dependencies as follows:

``` bash
pip install -r requirements.txt
```

Usage
---

``` bash
./UXTeamBugs.py {YYYY-MM-DD} {YYYY-MM-DD}
```
